我需要用 vibe coding 的方式寫一個 JavaScript 小遊戲，並且拆成三個檔案：index.html 負責網頁結構，style.css 負責遊戲畫面設計，script.js 負責遊戲邏輯。遊戲主題是美式賽車，玩家可以控制一台車子在道路上移動，並且要避開敵方車輛，同時收集金幣。
遊戲開始前要有規則畫面，規則內容需要顯示在一個清楚的區塊中，玩家可以透過按下 Start 按鈕或是空白鍵來開始遊戲。一旦遊戲開始，規則畫面就必須完全消失，不可以繼續疊在遊戲畫面上。
玩家的操作方式是透過鍵盤左右方向鍵來移動車子。玩家車子需要有清楚的造型，背景要呈現道路感。敵方車輛數量要適中，不要太多。金幣會隨機掉落，當玩家車子碰到金幣時需要加分，並且分數顯示要用金幣樣式來呈現，而不是單純的文字。
失敗條件是當玩家總共撞到五台敵方車輛時遊戲結束。遊戲結束後需要顯示 Game Over 畫面，並且展示分數。結束畫面中要能讓玩家選擇重新開始，重新開始時可以按 Start 按鈕或空白鍵，並且規則畫面不能再出現，同時分數要歸零，撞擊次數也要歸零。
畫面設計要有街機遊戲的感覺，規則頁面、分數區塊、以及 Game Over 畫面都要排版清楚，避免重疊，並且確保在遊戲流程中畫面顯示和隱藏的切換正確。
